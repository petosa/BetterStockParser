# BetterStockParser
<b>Command line interface (CLI) for parsing stock market data.</b>

Sucessor to previous project: http://github.com/petosa/Cross-Of-Gold

<ul>• Maintains a list of valid stock symbols in a MongoDB collection.</ul>
<ul>• Pulls raw stock data to a MongoDB collection.</ul>
<ul>• Parses stock data to trade signals. So far, there is support for X-day simple moving averages (SMAs) and intersection points.</ul>
<ul>• Displays stock data on a line graph.</ul>
